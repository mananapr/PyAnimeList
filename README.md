# PyAnimeList
PyAnimeList is a CLI Client for using [MyAnimeList](https://myanimelist.net/) written in Python 3

## Usage
Simply run `mal.py` as script:

    python3 mal.py

## Dependencies
- beautifulsoup4
- requests

Install dependencies using `pip`:

    pip install -r requirements.txt
